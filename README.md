# Fedora CoreOS Images


<img src="https://raw.githubusercontent.com/hauke-cloud/.github/main/resources/img/organisation-logo-small.png" alt="hauke.cloud logo" width="109" height="123" align="right">


This repository and its workflows build a Fedora CoreOS based on the configuration [fedora-coreos-config](https://github.com/hauke-cloud/fedora-coreos-config). Various flavors/packages are offered to support as many target systems as possible.

Here you will find:
- A Docker image that contains the iPXE images and provides them via a web server
- Various image files (from metal, metal4k to ISO files)

Once the repository is cloned, navigate to the directory:

